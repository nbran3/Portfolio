  • Census Project

    This was an in-depth project involving Census Data from the United States Census Bureau. I utilized SQL, Python, and PowerBI for this project. I then created a PowerBI presentation to summarize the SQL results. I then used python to do some data analysis in Python to find statistical relationships between the three different variables, median earnings, the proportion of college graudates, and the proportion of people with health insurance in each state.   • Baseball Project

    This was in depth data analysis project utizling baseball statistics from 1945-2015 utizling Python and Microsoft Excel. In Python I did a statistiscal analysis between the number of home runs per season and number of strikeout per season and found they were posistively correlated. Then I created a simple machine learning model via ARIMA to generate three seasons worth of Wins for each of the 30 MLB teams and then mocked a playoff like scenario for each season. I then visualized the standings and the playoffs for each season in Excel.
